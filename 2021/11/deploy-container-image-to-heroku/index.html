Deploy Container Image to Heroku

Nov 26, 2021

Status: DRAFT

Heroku

If you want to host your web application, Heroku is a great platform to host it. I started using Heroku when Docker wasn’t widely available in 2014. At that time, Heroku was a Platform as a service. Essentially, you provides your web application code, Heroku will build and deploy your application for you. Everything was managed as part of the git workflow.

Fast forward years later, Docker becomes the dominant platform for deploying your application code regardless of programming languages.
